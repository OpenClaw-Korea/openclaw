# OpenClaw 입문 가이드

## OpenClaw이란?

OpenClaw은 **멀티채널 AI 게이트웨이 플랫폼**입니다. 다양한 메시징 채널(WhatsApp, Telegram, Discord, iMessage, Slack, Signal 등)을 AI 모델(Claude, OpenAI 등)에 연결하여, 하나의 개인 AI 비서를 여러 채널에서 동시에 사용할 수 있게 해줍니다.

TypeScript/Node.js 기반의 모노레포 프로젝트로, pnpm 워크스페이스를 활용하여 구성되어 있습니다.

- OpenClaw 프로젝트의 전체 구조를 파악하고 싶은 신규 기여자
- TypeScript 모노레포 프로젝트의 실제 사례를 학습하고 싶은 개발자
- 메시징 채널과 AI 모델을 연동하는 아키텍처에 관심 있는 개발자

## 사전 준비사항

OpenClaw 프로젝트를 로컬에서 실행하고 개발하려면 다음 환경이 필요합니다.

| 항목           | 요구사항  | 설명                                    |
| -------------- | --------- | --------------------------------------- |
| **Node.js**    | v22 이상  | ESM 모듈과 최신 API를 사용합니다        |
| **pnpm**       | 최신 버전 | 모노레포 워크스페이스 관리에 사용됩니다 |
| **TypeScript** | 5.9+      | strict 모드와 ESM을 기본으로 사용합니다 |
| **Git**        | 최신 버전 | 소스 코드 관리 및 협업에 필요합니다     |

### 기본 지식

다음 기술에 대한 기본적인 이해가 있으면 학습이 수월합니다.

- **TypeScript**: 타입 시스템, ESM `import`/`export`, `async`/`await` 등 기본 문법
- **Node.js**: 이벤트 루프, 모듈 시스템, 파일 시스템 API
- **Express**: HTTP 서버, 미들웨어, 라우팅 개념
- **CLI 도구**: Commander.js 기반 명령행 인터페이스 패턴

### 빠른 시작

프로젝트를 클론하고 의존성을 설치하는 기본 과정입니다.

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
pnpm install
pnpm build
```

빌드 도구로 `tsdown`을 사용하며, 테스트 프레임워크는 `vitest`입니다. 엔트리 포인트는 `openclaw.mjs` 파일이며, 여기서 `src/entry.ts`를 거쳐 CLI 프로그램이 시작됩니다.
